# modulo-autenticacion-SneyderyYorman

Historia 1 – Login

Como usuario registrado
quiero iniciar sesión con mi correo y contraseña
para acceder al dashboard de la aplicación.

Historia 2 – Registro

Como nuevo usuario
quiero crear una cuenta registrando mis datos
para poder ingresar al sistema.

Historia 3 – Dashboard

Como usuario autenticado
quiero ver un dashboard con mi información
para gestionar mis datos dentro de la plataforma.
